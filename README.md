# WorkoutPAL
Un assistant d'entraînement pour suivre votre progression pendant votre séance d'entraînement.

## Fonctionnalités
- ✅ 6 séances prédéfinies (Abdominaux, Pectoraux, Avant-bras, Biceps, Épaules, Dos)
- ✅ Gestion des exercices, séries et répétitions
- ✅ Timer de repos
- ✅ Mode édition
- ✅ Fonctionne hors ligne
- ✅ Sauvegarde locale, aucune données partagées
- ✅ Installation en mode PWA (option navigateur -> Ajouter à l'écran d'accueil)
